# 🛍️ La Moda – Full-Stack Fashion Aggregator Platform

🌐 **Live Demo:** https://la-moda.onrender.com

La Moda is a full-stack fashion aggregation platform that brings together products from multiple e-commerce websites like Amazon, Myntra, Flipkart, and more — with advanced shopping features and a powerful admin analytics dashboard.

---

## 🚀 Key Highlights

* Full-stack e-commerce architecture
* Integrated payment gateway (Cashfree)
* Advanced admin analytics dashboard
* Real-time product interaction system

## ✨ User Features

* 🔍 Live Search for instant product discovery

* 🛒 Add to Cart & Wishlist (Login required)

* 💳 Multiple Payment Options (Cashfree Integration):

  * UPI
  * Credit/Debit Card
  * Net Banking
  * EMI
  * Cash on Delivery

* 🧠 Smart Recommendations:

  * Recently Viewed
  * Products You Might Like

* 🎟️ Coupons & Order Management

* ⭐ Ratings & Reviews System

* ⏳ Flash Sales with Countdown Timer

* 📏 Size Selection with Size Chart

* 📦 Stock Indicator (In Stock / Low Stock / Out of Stock)

* 💰 Dynamic Pricing (Old vs New Price)

* 🔐 Secure Authentication System

* 🧭 Category & Subcategory Navigation

## ⚙️ How to Run Locally

1. Clone the repository

   ```
   git clone https://github.com/yourusername/la-moda
   ```

2. Navigate to project folder

   ```
   cd la-moda
   ```

3. Start XAMPP / WAMP server

4. Ensure MongoDB server is running

5. Configure database connection

6. Run in browser

   ```
   http://localhost/la-moda
   ```

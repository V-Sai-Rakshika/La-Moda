## Overview

La Moda is a full-stack fashion aggregation platform that brings together products from multiple e-commerce providers like Amazon, Myntra, and Flipkart into a unified interface.

It goes beyond a typical e-commerce clone by focusing on **product discovery, comparison, and analytics**, while also providing a powerful admin dashboard for tracking user behavior, sales, and trends.

The platform is designed to simulate a real-world scalable shopping system with dynamic data, user interactions, and integrated payments.

## Features

| Feature                    | Description                                         |
| -------------------------- | --------------------------------------------------- |
| **Live Product Search**    | Real-time search across aggregated product listings |
| **Product Aggregation**    | Unified browsing experience from multiple platforms |
| **Detailed Product Pages** | Pricing, variants, stock status, and descriptions   |
| **Shopping Cart System**   | Add, remove, and manage cart items                  |
| **Wishlist System**        | Save products for later (login required)            |
| **Authentication**         | Secure user login & session handling                |
| **Smart Recommendations**  | Recently viewed & suggested products                |
| **Dynamic Pricing UI**     | Old vs new price with discount visualization        |
| **Flash Sales**            | Time-based offers with countdown timers             |
| **Ratings & Reviews**      | User-generated feedback system                      |
| **Coupon System**          | Apply discount codes during checkout                |

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/la-moda
cd la-moda
```

### 2. Start local server

* Use XAMPP / WAMP for PHP
* Ensure MongoDB is running

### 3. Configure database

* Set connection details in backend files

### 4. Run the project

```
http://localhost/la-moda
```
